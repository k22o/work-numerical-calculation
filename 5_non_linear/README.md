# 非線形方程式

方程式は、線型方程式 ($ax+b=0$) と非線形方程式に分類される。
非線形方程式のうち、$\sum a_i x^i=0$ は代数方程式と呼ばれる。
代数方程式以外の非線形方程式 (狭義の非線形方程式) は、解の存在や一意性が保証されない。

## 1次元方程式のニュートン法

非線形方程式 $f(x)=0$ の解を求める。

初期値$x_0$に対して、k=0,1,2 ...と繰り返して

1. $x_{k+1} := x_k - \frac{f(x_k)}{f\prime(x_k)}$ 
2. 収束判定

で求める方法をNewton法と呼ぶ。

$f(x)$が$x$近傍で2階以上微分可能な場合、Taylor展開において、$h_0=x_1-x_0$ と定義して、

$f(x_1) = f(x_0) + f\prime(x_0)h_0 +O(h_o^2)$

と定義できる。本来は、$f(x_0+h_0) = 0$ となる値を探すが、$f(x_0) + f\prime(x_0)h_0 = 0$ となる $x_1$を採用すれば、$f(x_1)=O(h_o^2)$
となる。計算を繰り返し、$O(h_o^2)$が十分小さくなれば、$f(x)=0$ が求まる (二次収束)。
