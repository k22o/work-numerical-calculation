#  数値計算

## 内容

- 1_basic_concept: 数値計算に共通する基本概念
- 2_basic_calculation: 簡単な計算をいくつか
- 3_linear_system_of_equations: 連立1次方程式
- 4_matrix: 行列
- 5_non_linear: 非線形方程式
- 6_interpolation_lsm: 補完・最小二乗法
- 7_calculus: 微積分
- 8_ode: 状微分方程式
- 9_pde: 偏微分方程式

## 環境

Python がインストール済みであることを前提とする。

### 実行

```bash
uv run python3 xxxx
```
