# 連立一次方程式

連立一次方程式を、$A\bm{x}=\bm{b}$ と定義する。

- 直接法: Aを直接変形して、有限会の操作でxを求める
- 反復法: 
- クリロフ部分空間法

## 直接法

### ガウスの消去法 (吐き出し法) による解法

線形代数で行われるガウス消去法をそのまま実装することで数値計算は可能。
ただし、計算量の観点で後述する手法の方が効率的である。

### LU分解による解法

LU分解を用いて、$A=LU$ の形に分解する。
 - L: 下三角行列
 - U: 上三角行列

もとの演算を以下の2つに分解してそれぞれ計算する。それぞれ、全進代入/後進代入と呼ぶ。
 - $L\bm{y}=\bm{b}$
 - $L\bm{x}=\bm{y}$

### LDU分解による解法

対角行列$D$を用いて、$A=LDU$ とする分解をLDU分解と呼ぶ。
これを用いても連立一次方程式を解くことができる。

(LU分解ではUが一意に定まらないが、LDU分解では一意に定まる)

### ガウス・ジョルダン法

ガウスの消去法を拡張した方式。
連立一次方程式の解法としても使えるが、一般には逆行列の計算に利用される (通常は、数値計算ライブラリの関数で逆行列を求める)。

逆行列が求まれば、$x=A^{-1}b$ として解が求まる。

## クラウト法

本質的にはLU分解と同じで、途中の計算方法が異なるだけ。

#### 修正コレスキー分解

$A$が実対称行列 ($A=A^T$) の場合は、LDU分解した際に、$U=L^T$ となる。

すなわち、上三角 or 下三角を求めるだけで済む。これを、修正コレスキー分解と呼ぶ。
