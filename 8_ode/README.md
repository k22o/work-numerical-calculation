# 常微分方程式

## 常微分方程式について

一般に、$\phi(x, \bm{y}, \frac{d\bm{y}}{dx}, ... \frac{d^r\bm{y}}{d^rx}) = 0$ で定義される。

これは、n次元1階微分方程式と同一視することができて、$\frac{d\bm{y}}{dx} = \bm{f}(x,\bm{y})$ ととらえることができる。

ここでは、初期値 $\bm{y}(x_0) = y_0$ の与えられた、初期値問題 (initial value problem; IVP) について考える。
IVPが唯一の解をもつためには、リプシッツ条件を満たす必要がある。

## 数値解析の方法

いずれも「現在の点から次の点を予測して進む」一段法。精度の違いは1ステップあたりの関数評価回数と打ち切り誤差の次数に表れる。

| 手法 | 精度 | 関数評価回数/ステップ |
|------|------|----------------------|
| オイラー法 | 1次 | 1 |
| 中点法 | 2次 | 2 |
| ルンゲクッタ4次 | 4次 | 4 |

### オイラー法

最もシンプルな1次精度の手法。現在の勾配をそのまま使って次の点を予測する。

$$y_{n+1} = y_n + h \, f(x_n, y_n)$$

### 中点法

オイラー法でステップの半分だけ進めた中間点での勾配を使う2次精度の手法（2次ルンゲクッタ法の一種）。

$$k_1 = f(x_n, y_n)$$

$$k_2 = f\!\left(x_n + \frac{h}{2},\; y_n + \frac{h}{2} k_1\right)$$

$$y_{n+1} = y_n + h \, k_2$$

### ルンゲクッタ法

最も広く使われる4次精度の手法。4点の勾配を重み付き平均して次の点を求める。

$$k_1 = f(x_n, y_n)$$

$$k_2 = f\!\left(x_n + \frac{h}{2},\; y_n + \frac{h}{2} k_1\right)$$

$$k_3 = f\!\left(x_n + \frac{h}{2},\; y_n + \frac{h}{2} k_2\right)$$

$$k_4 = f(x_n + h,\; y_n + h \, k_3)$$

$$y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$
