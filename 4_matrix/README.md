# 行列関係

## 固有値・固有ベクトル (eigenvalue, eigenvector)

### 原則

$A\bm{v} = \lambda\bm{v}$ となる値 $\lambda$ が固有値である (定義)。

行列式 $det(A-\lambda I) =0$ となる $\lambda$ を求めればよい (定理)。

行列式は、4次元までは有限回の代数操作で求められるが、5次元以上だと有限回で真値は求められない。よって、一定の段階で打ち切ることを前提としたアルゴリズムが利用される。

### アルゴリズム

- 固有値と固有ベクトルを同時に求める方法
   - べき乗法
   - 逆べき乗法
   - ヤコビ法
- 固有値のみを求める方法
   - QR法
   - Bisection法

### リダクション

計算量を減らすために、相似変換による、行列の疎行列化を行うことがある。これをリダクションと呼ぶ。
相似変換とは、正則行列 $P$ に対して、

$det(P^{-1}AP-\lambda I) = 0$ 

を利用して固有値を求められる。固有ベクトルは $P^{-1}\bm{v}$で求まる。

適切な$P$を選ぶことで、行列を疎行列化して計算を簡略化することができる。こうした $P$や$P^{-1}AP$を求めるアルゴリズムの１つがHouseholder法である。
